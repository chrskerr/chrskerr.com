WeVote

  • Link to site
  • GitHub repo

This was our final assignment. It is a VueJS frontend on Node/Express/Graphql/SQL backend.

The interface is a basic voting application, with some unique voter logic which then saves votes into a blockchain-style immutable SQL database. A visualisation of this process can be seen on the Analytics tab, currently pretty raw for debugging purposes but this will be cleaned up into voter analytics later.

The intention is to create a cheap, anonymous, auditable, immutable referendum mechanism that Australia can rely upon to influence policy and create the nation that we choose to be.
